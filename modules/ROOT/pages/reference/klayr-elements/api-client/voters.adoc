This is a resource for interacting with the `voters` endpoint provided by the Klayr public API.
Each of the following methods can be accessed via the `voters` property of an `APIClient` instance.

== get

This returns all votes received by a delegate.

=== Syntax

[source,js]
----
get(options)
----

=== Parameters

`options`: Please see the options in the following link: xref:{url_klayr_core_api}[Core API documentation^].

=== Return value

`Promise`: Resolves to an API response object.

=== Examples

[source,js]
----
client.voters.get({ username: 'oliver' })
    .then(res => {
        console.log(res.data);
    })
----
