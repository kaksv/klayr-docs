@klayr/tree is a package which contains Merkle tree implementations for use with Klayr-related software.

== Installation

[source,bash]
----
npm install @klayr/tree
----

== Update

To perform an upgrade, execute the following command:

[source,bash]
----
npm update @klayr/tree
----
